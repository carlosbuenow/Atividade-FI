Hardware

Placa-Mãe

Componente que acopla todos os outros componentes de um computador, responsável por fazer a ligação entre as peças. Referente ao chipset, são projetados para Intel e AMD, atualmente.

Processador / CPU

Componente essencial e responsável por fazer todo o processamento e a execução do computador, composto por micro núcleos (físicos) e threads (núcleos virtuais). Quanto mais, melhor. A velocidade desses núcleos é ditada pelo clock, que é medido por Gigahertz. Entre as fabricantes atuais de CPUs para DESKTOP temos apenas Intel e AMD.

Placa de Vídeo

Componente responsável pela reprodução de vídeo e imagem na tela do computador. Necessária para tarefas pesadas, como jogos, edição de vídeo e renderização 3D. Entre as fabricantes de chips para placa de vídeo temos hoje, no mercado, apenas Intel, AMD e Nvidia, com propostas semelhantes, contendo produtos low-profile até High-End.

Memória Ram

Memória responsável por armazenar dados temporários do sistema, dados esses que são apagados após o fim da execução de determinada tarefa. A velocidade dessas memórias é medida por MegaTransfer. Quanto maior, melhor, sendo o modelo geracional classificado como DDR. Atualmente, a geração mais alta é a DDR5. A velocidade da memória pode ser limitada pelo processador.

HD / SSD

Disco rígido responsável por armazenar dados permanentes. A média de velocidade é descrita como "leitura" e "gravação", sendo os SSDs um salto tecnológico extremo dos HDs. Dispositivos de entrada e saída: Entrada é tudo que compõe os periféricos, sendo o mais comum o teclado, com a função de dar entrada para o usuário digitar comandos no sistema. Entre os dispositivos de saída, temos como destaque o monitor e caixas de som, reproduzindo imagem e som, respectivamente.

Evolução do Hardware

A evolução do hardware tem sido marcada por uma busca constante por maior velocidade, eficiência energética e miniaturização.

  • Processadores saíram de válvulas e transistores discretos para milhões de transistores em um único chip, seguindo a Lei de Moore, que prevê o dobramento da densidade dos transistores a cada dois anos, resultando em mais núcleos e clocks mais rápidos.
  • As Memórias RAM evoluíram de padrões lentos e de baixa capacidade (como as DDR iniciais) para as atuais DDR5, com frequências altíssimas e latências cada vez menores, acompanhando a demanda dos processadores.
  • O armazenamento permanente deu um salto com a transição de HDs mecânicos para SSDs baseados em memória flash, eliminando peças móveis e acelerando dramaticamente o tempo de leitura e gravação.
  • As Placas de vídeo (GPUs) deixaram de ser simples adaptadores gráficos para se tornarem processadores paralelos massivos, essenciais não só para jogos e renderização, mas também para computação de alto desempenho e inteligência artificial. Essa evolução permite que os computadores atuais realizem tarefas inimaginavelmente complexas para as máquinas de décadas atrás, tornando o hardware cada vez mais poderoso e acessível.

O que é um Sistema Operacional?

Um sistema operacional é o software que gerencia diretamente o hardware e os recursos de um sistema, como a CPU, a memória e o armazenamento. O sistema operacional está posicionado entre aplicações e hardwares e faz as conexões entre todos os seus softwares e recursos físicos que realizam o trabalho.

O que é o Unix?

O Unix é um sistema operacional de código fechado lançado em 1971 que trouxe novos conceitos a este tipo de software, como arquivos de dispositivo e processos de computadores para o mercado. Ele marcou a história ao servir de base para a criação de outras plataformas populares, como o Linux e o macOS, e eventualmente ganhou o título de “pai dos sistemas operacionais”.

Como o Unix funciona?

Atua como um sistema operacional que gerencia a interação entre o hardware e o usuário através de um núcleo e programas de sistema. Ele possui uma estrutura de sistema de arquivos hierárquico em forma de árvore e oferece multitarefa e multiusuário, com cada usuário tendo permissões e direitos sobre os arquivos.

  • Nucleo

    • A parte central do sistema operacional, responsável por gerenciar o hardware, agendar processos e controlar a memória. 

  • Programas de Sistemas

    • Incluem o shell (intérprete de comandos) e utilitários, que servem como interface para os usuários e para o núcleo do sistema. 

  • Sistemas de Arquivos Hierarquicos

    • Organiza os diretórios e arquivos em uma estrutura em forma de árvore, com o diretório raiz (/) no topo, facilitando a localização de dados. 

O que é o Linux?

Linux® é um sistema operacional open source criado por Linus Torvalds em 1991.

 

Como o Linux Funciona?

O Linux foi desenvolvido para ser semelhante ao UNIX, mas evoluiu e passou a ser executado em uma grande variedade de hardwares, de telefones a supercomputadores. Todo sistema operacional baseado em Linux inclui o kernel do Linux, que gerencia recursos de hardware, e um conjunto de pacotes de software que compõem o restante do sistema operacional. 

Quem usa o Linux?

Os usuários do Linux aderem a este sistema operacional em razão de seus recursos de segurança e versatilidade, entre outros motivos.

  • Kernel

    • É o componente base do sistema operacional. Sem ele, o sistema operacional não funciona. O kernel gerencia os recursos do sistema e se comunica com o hardware.

  • Espaço do usuário do sistema

    • É a camada administrativa para tarefas no nível do sistema, como configuração e instalação de software.

  • Aplicações

    • São um tipo de software que permite a realização de uma tarefa. As aplicações incluem tudo, desde ferramentas de desktop e linguagens de programação até conjuntos de soluções empresariais com vários usuários.

O que é open source?

É um sistema operacional gratuito Qualquer pessoa pode executar, estudar, modificar e redistribuir o código-fonte, ou ainda vender cópias do código modificado, desde que isso seja feito sob a mesma licença.

O que é o Windows?

O Windows é um sistema operacional desenvolvido pela Microsoft que alcançou grande popularidade e dominou o mercado para computadores pessoais ao longo de sua história. Desde 1985, várias versões do Windows foram lançadas — com maior ou menor sucesso — e incluem famílias e subfamílias de sistemas que atendem não apenas consumidores, mas também diversos setores da indústria de computação.

Como Funciona o Windows?

O Microsoft Windows é um tipo de sistema operacional comumente usado em computadores e laptops. É responsável por gerenciar o hardware e o software relacionados ao seu computador e fornece uma interface que permite interagir com o seu dispositivo. O Windows funciona fornecendo uma interface gráfica do usuário na qual você pode abrir programas, navegar por menus e configurações, além de acessar diferentes recursos, como rede, impressão, reprodução de áudio/vídeo e muito mais. Além disso, ele também lida com tarefas como gerenciamento de memória e agendamento de recursos em segundo plano, para que seu computador seja executado com eficiência. Em suma, o Microsoft Windows é uma parte essencial de como a maioria dos computadores funciona hoje.

O que é DOS?

O DOS é um sistema operacional de disco que gerencia o armazenamento e a recuperação de dados em um disco rígido. Ele fornece uma interface entre o hardware do computador e os programas de software, permitindo que os usuários executem aplicativos e acessem arquivos armazenados no disco. O DOS é um sistema operacional de linha de comando, o que significa que os usuários interagem com ele digitando comandos de texto em vez de usar uma interface gráfica.

Como funciona o DOS?

O DOS funciona lendo e gravando dados em um disco rígido por meio de comandos específicos. Ele organiza os arquivos em diretórios e subdiretórios para facilitar a localização e recuperação de dados. O DOS também gerencia a memória do computador, permitindo que os programas sejam carregados e executados de forma eficiente. Ele controla o acesso ao hardware do computador, como o teclado, o monitor e os dispositivos de armazenamento.

O que é sistema operacional Android?

O Android OS é um sistema operacional móvel, desenvolvido pelo Google, e baseado no núcleo Linux. O Android é, portanto, responsável por gerenciar e executar todos os recursos de hardware e software em um dispositivo, o que permite que os aplicativos interajam com o hardware. Ele foi projetado principalmente para dispositivos móveis com touchscreen, como smartphones e tablets.

Como funciona o Android?

A principal função do Android é fornecer um ambiente operacional completo para dispositivos móveis. Ele gerencia o hardware do dispositivo, como processadores, memória, tela sensível ao toque, câmera, sensores, conectividade e muito mais. O Android fornece uma interface de usuário intuitiva e interativa, permitindo que os usuários acessem aplicativos, serviços e recursos do dispositivo.

    Recursos importantes do Android

  • Código aberto

    • O Android tem código aberto, ou seja, os fabricantes podem adaptá-lo aos seus dispositivos. Por ter essa natureza de código aberto, surgem várias oportunidades, incluindo flexibilidade, personalização e desenvolvimento pela comunidade. Isso permite a implementação de melhorias contínuas e ideias inovadoras, impulsionadas por uma rede global de desenvolvedores.

  • Ecossistema de apps enorme

    • A Google Play Store oferece acesso a milhões de aplicativos para tudo: de produtividade e comunicação a jogos e entretenimento. Assim, você com certeza encontrará o aplicativo perfeito para suas necessidades.

  • Integração de Serviços Google

    • O Android facilita sua vida ao integrar perfeitamente os serviços do Google, como Gmail, Google Maps e outros aplicativos, deixando as informações e tarefas acessíveis. Ele também permite acessar seus dados de forma simples e ficar conectado entre dispositivos.

  • Personalização

    • A interface do usuário do Android é altamente personalizável. Você pode personalizar seus dispositivos com widgets, lançadores e temas para criar uma experiência que reflita verdadeiramente seu estilo. Além disso, recursos inovadores como o Google Assistente ajudam você a executar tarefas sem usar as mãos.

  • Acessibilidade

    • O Android é uma plataforma projetada para todos, por isso, você pode usar vários recursos de acessibilidade que adaptam a plataforma para pessoas com deficiência, incluindo leitores de tela, TalkBack e outras tecnologias assistivas.

A Evolução dos Softwares de Desenvolvimento

Desenvolvimento de software é o processo de criar, projetar, codificar, testar, implantar e manter programas digitais que resolvem problemas e automatizam tarefas. Seu ciclo de vida envolve análise de requisitos, planejamento, design, implementação, testes, implantação e manutenção contínua. Ele se apoia em linguagens como Python, Java e JavaScript, em metodologias ágeis como Scrum e Kanban e em práticas de segurança como DevSecOps. A origem remonta ao século XIX com Ada Lovelace, mas ganhou força a partir da década de 1970 com os microcomputadores, a comercialização pela IBM e sistemas operacionais como o MS-DOS. As principais aplicações incluem desenvolvimento web e mobile, inteligência artificial, análise de dados, sistemas de gestão (ERP), plataformas no-code e tecnologias de infraestrutura como Docker e Kubernetes. Os tipos de software englobam aplicações, sistemas operacionais, ferramentas de desenvolvimento e softwares embarcados. Entre as tecnologias populares, destacam-se linguagens como Python, JavaScript e Java, e frameworks como React e Flutter.

1950-1960

Primeiras Linguagens de Alto Nível e a Abstração do Hardware

Contexto Histórico

Nesse período, programar significava trabalhar diretamente com códigos binários (0 e 1) ou assembly, exigindo profundo conhecimento do funcionamento interno da máquina. Essa barreira técnica limitava o número de pessoas capazes de programar, restringindo o acesso ao desenvolvimento de software.

Impacto:

Essas inovações diminuíram a complexidade da programação, abriram espaço para profissionais de outras áreas e introduziram a noção de abstração — conceito que guiaria toda a evolução futura do software.

    Evoluções Tecnológicas

  • O FORTRAN (1957) introduziu a ideia de expressar cálculos matemáticos de maneira mais intuitiva, sendo amplamente usado em ciências e engenharia.

  • O COBOL (1959) aproximou a programação da linguagem natural, facilitando a criação de sistemas corporativos e tornando-se referência em bancos e governos.

  • O LISP (1958) inovou ao adotar listas como estrutura fundamental, influenciando o desenvolvimento de inteligência artificial e linguagens funcionais modernas.

    • A invenção do compilador permitiu que programas escritos em linguagens de alto nível fossem traduzidos automaticamente para código de máquina, acelerando o desenvolvimento.

1970–1980

Consolidação, Linguagens Portáveis e Metodologias Tradicionais

Contexto Histórico

Com sistemas cada vez mais complexos, empresas e governos passaram a demandar softwares confiáveis e padronizados. Surgiu a necessidade de metodologias formais para organizar o processo de desenvolvimento.

    Evoluções Tecnológicas

  • A linguagem C (Dennis Ritchie, 1972) foi um divisor de águas: rápida, flexível e portável, influenciou a construção de inúmeros softwares.

  • O Pascal e outras linguagens acadêmicas reforçaram a importância da programação estruturada, que buscava reduzir erros e facilitar a manutenção.

1980–1990

Interfaces Gráficas, Ferramentas de Programação Visual e Usabilidade

Contexto Histórico

O software passou a ser desenvolvido também para usuários não técnicos, fazendo surgir a preocupação com usabilidade e acessibilidade.

    Evoluções Tecnológicas

  • As GUIs (Interfaces Gráficas) substituíram linhas de comando por janelas, ícones e menus.

  • Ferramentas como Visual Basic e Delphi popularizaram a programação visual, permitindo que desenvolvedores criassem sistemas arrastando componentes.

  • Softwares de produtividade (Word, Excel, Lotus 1-2-3, CorelDRAW) tornaram-se essenciais para o dia a dia.

1990–2000

A Internet e a Revolução do Desenvolvimento Ágil

Contexto Histórico

Com a popularização da internet, os softwares passaram a ser distribuídos em escala global e usados em tempo real. Isso trouxe novos desafios: interconectividade, segurança e necessidade de respostas rápidas às mudanças.

    Evoluções Tecnológicas

  • HTML, CSS, JavaScript possibilitaram páginas interativas e o nascimento das aplicações web.

  • PHP, ASP e Java permitiram sistemas dinâmicos, como fóruns, e-commerces e portais.

  • O movimento de software livre fortaleceu a colaboração global entre desenvolvedores.

2000

Computação em Nuvem, DevOps e Inteligência Artificial

Contexto Atual

O software está em todo lugar: em serviços digitais, plataformas de comunicação, sistemas corporativos, redes sociais e aplicações inteligentes. A complexidade cresceu, mas também surgiram ferramentas para lidar com ela.

  • Computação em nuvem: plataformas como AWS, Azure e Google Cloud eliminaram barreiras de infraestrutura, permitindo escalabilidade global.

  • DevOps: integra desenvolvimento e operações, com CI/CD, Docker e Kubernetes, acelerando entregas.

  • Inteligência Artificial: machine learning e deep learning transformaram software em sistemas capazes de aprender e automatizar decisões.
